	### How to distinguish medals ###
	# Each medal has a 4 digit code at the end of it. Each digit with value of either 0 or 1 represents whether the type of upgrade is present.
	# If value is 1, the upgrade is present
	# All medals follow the nameing scheme RUS_TOZ_Medal_XXXX where each X is a value that is either 0 or 1
	# e.g. RUS_TOZ_Medal_0110 represents a TOZ medal with the Defence and organisation upgrades.
	# First digit = attack medal upgrade
	# Second digit = Defence medal upgrade
	# Third Digit = organisation medal upgrade
	# Fourth Digit = breakthrough medal upgrade
